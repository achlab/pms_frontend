Clear Authentication Data

This tool will clear all authentication data (localStorage, sessionStorage, cookies) to break out of any authentication loops.

What this does:

  • Clears all localStorage data
  • Clears all sessionStorage data
  • Clears all authentication-related cookies
  • Reloads the page

Instructions:

  1. Click "Clear Auth Data"
  2. Wait for the success message
  3. Click "Go to Login" or "Go to Register"
  4. Try registering again
